Install steps

  • Mac: Open the .dmg, drag Vitae into Applications, launch from Applications.
  • Linux: sudo dpkg -i vitae-health-tracker_*.deb then launch from your app menu.

Browser-only option

No install needed — open index.html directly in Firefox, Safari, or any modern browser.

For developers — publish a release

  • Set githubRepo in release-config.json (see release-config.example.json).
  • Push this project to GitHub, then tag a version: git tag v1.0.0 && git push origin v1.0.0
  • GitHub Actions builds both installers and attaches them to the release automatically.
