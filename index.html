Help & Source References

BMI Calculation

Body Mass Index is calculated as:

BMI = weight (kg) ÷ [height (m)]²

Imperial conversions use NIST standard factors: 1 lb = 0.45359237 kg, 1 in = 2.54 cm (exact). All BMI values are computed in metric before categorization.

WHO / CDC BMI Categories (Adults ≥20)

  • Underweight: BMI < 18.5
  • Healthy Weight: BMI 18.5 – 24.9
  • Overweight: BMI 25.0 – 29.9
  • Obesity Class I: BMI 30.0 – 34.9
  • Obesity Class II: BMI 35.0 – 39.9
  • Obesity Class III (Severe): BMI ≥ 40.0

Lifespan & Mortality Data

Life expectancy figures shown are global estimates from the World Health Organization and UN World Population Prospects (2023 revision). Infant mortality rates are per 1,000 live births. Elderly figures show remaining life expectancy at age 65 and 80.

WHO — Global Health Observatory: Mortality & Life Expectancy UN DESA — World Population Prospects 2024 CDC NCHS — FastStats: Life Expectancy (United States) WHO — Under-Five Mortality Rate Indicator

How to Run

Open index.html in any modern browser — double-click the file, drag it into a browser window, or run open bmi-health-tracker/index.html in Terminal. No installation or server required.

Data Storage & Persistence

Saved entries are written to localStorage and IndexedDB simultaneously. Data persists after closing the tab, quitting the browser, or a browser crash. Draft notes/metrics auto-save while typing. Use Export CSV for off-device backup. Use Import CSV to restore.

Age-Based Life Stages & BMI Scales

Set your birth date and sex in the Profile panel. The app calculates your age on each entry date and selects the correct reference scale:

  • Infant (0–23 months): WHO Child Growth Standards — weight-for-length; BMI percentiles begin at age 2
  • Child (2–12 years): CDC BMI-for-age sex-specific percentiles
  • Adolescent (13–19 years): CDC BMI-for-age sex-specific percentiles
  • Adult (20–64 years): WHO/CDC fixed BMI cutoffs
  • Mature Adult (65+ years): WHO/CDC adult BMI + geriatric context

Pediatric categories: Underweight (<5th %ile), Healthy (5th–84th), Overweight (85th–94th), Obesity (≥95th). Data from CDC 2000 Growth Charts.

CSV File Format

Columns: date, weight_kg, weight_lb, height_cm, height_ft, height_in, bmi, category, life_stage, age_at_entry, bmi_percentile, unit_system, notes, recorded_at

Import accepts date plus any weight column (weight_kg or weight_lb) and any height columns (height_cm or height_ft + height_in). Empty imperial or metric fields are fine if the other unit is filled in.

Install as Desktop App (recommended if you don't use Chrome/Edge)

Native installers use your system's built-in web engine (Safari WebKit on Mac, WebKitGTK on Linux) — not Chrome.

Download page — Mac .dmg and Linux Mint .deb from GitHub Releases.

  • Automated builds: Push tag v1.0.0 to GitHub → Actions builds both installers
  • Build locally: ./build-mac.sh or ./build-linux.sh

Set githubRepo in release-config.json so the download page finds your releases.

Browser Install (PWA — Chrome/Edge only)

Run python3 -m http.server 8080, open localhost:8080, then Install. Works offline after first load.

Consult a healthcare provider for clinical assessment. Pediatric and elderly individuals may require specialized evaluation beyond standard adult BMI categories.
